# G27 Oracle Trading System Skill

## Description

G27 Oracle is an autonomous cryptocurrency trading system with AI-powered decision making. It manages your entire portfolio by analyzing each holding with Mirofish Monte Carlo simulation and Oracle AI decision matrix.

## Features

- **Portfolio-wide Management**: Scans and manages all holdings
- **Per-Coin Strategies**: Each coin has unique RSI thresholds, stop-loss, and take-profit parameters
- **Major vs Meme Differentiation**: Conservative strategies for major coins, aggressive for meme coins
- **Mirofish Simulation**: 500-agent Monte Carlo simulation for signal validation
- **Oracle Decision Matrix**: AI-powered buy/sell/hold decisions
