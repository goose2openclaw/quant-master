## 二、每日流程 (两条线)

### 2.1 高频量化线 (自适应渐进式监控)

```
┌─────────────────────────────────────────────────────────────┐
│              高频量化线 (自适应渐进式)                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [每15分钟]                                               │
│  ┌─────────────┐                                          │
│  │ 1. 市场扫描 │ → 全部币种异动检测                         │
│  └──────┬──────┘                                          │
│         ↓                                                   │
│  [发现异动]                                                 │
│  ┌─────────────┐                                          │
│  │ 2. 定焦     │ → 锁定异动币种                            │
│  └──────┬──────┘                                          │
│         ↓                                                   │
│  📊 自适应渐进式监控:                                       │
│  ┌─────────────────────────────────────────────────────┐  │
│  │ 阶段1: 5次 × 500ms   = 2.5秒   (快速扫描)           │  │
│  │ 阶段2: 10次 × 5秒    = 50秒    (短期确认)           │  │
│  │ 阶段3: 10次 × 15秒   = 150秒   (中期验证)            │  │
│  │ 阶段4: 5次 × 30秒    = 150秒   (稳定观察)            │  │
│  │ 阶段5: 5次 × 60秒    = 300秒   (趋势确认)            │  │
│  │ 阶段6: ∞ × 5分钟    = 长期监控                       │  │
│  └─────────────────────────────────────────────────────┘  │
│         ↓                                                   │
│  ⚡ 触发条件 (任意满足即执行):                              │
│  ├── 异常波动 (>3%) → 紧急平仓                            │
│  ├── 趋势反转 → 反向建仓                                   │
│  ├── 成交量爆发 (>3倍) → 顺势建仓                         │
│  ├── RSI超买(>80) → 平仓                                  │
│  ├── RSI超卖(<20) → 建仓                                  │
│  └── 价差异常(>0.5%) → 观望                               │
│         ↓                                                   │
│  📊 置信度计算: 趋势+稳定性+成交量                         │
│         ↓                                                   │
│  🏷️ 决策: ≥8分强烈建议 → 自动执行                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 三、决策级别

### 3.1 高频线级别

| 置信度 | 建议 | 操作 |
|--------|------|------|
| ≥8分 | 强烈建议 | 自动建仓/平仓 |
| 6-8分 | 建议 | 观察/手动 |
| 4-6分 | 观望 | 不操作 |
| <4分 | 忽略 | 不操作 |

### 3.2 追踪线级别

| 置信度 | 建议 | 操作 |
|--------|------|------|
| ≥8分 | 强烈建议 | 自动加仓/减仓/平仓 |
| 6-8分 | 建议持有 | 继续观察 |
| 4-6分 | 建议减仓 | 部分止盈 |
| <4分 | 建议止损 | 全部平仓 |
