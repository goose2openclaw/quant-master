# go-core - 加密量化预测核心引擎

## 概述
go-core 是加密货币量化预测的核心引擎，蒸馏 Mirofish 1000 智能体仿真，结合多维度专业分析子技能，形成统一预测输出。

## 🎯 设计理念

```
Mirofish 1000智能体 (核心)
        ↓
   投票共识 + 贝叶斯更新
        ↓
多维度分析 (量子/热力/人性/机构)
        ↓
加权组合优化 (go-ensemble)
        ↓
自主迭代进化 (go-meta)
        ↓
   最终预测输出
```

## 🚀 API

### Python
```python
from skills.go_core import GoCore

# 初始化
go = GoCore(num_agents=1000)

# 预测
result = go.predict('BTC', interval='1h', period='90d')
print(f"信号: {result.signal}")
print(f"置信度: {result.confidence:.0%}")
print(f"理由: {result.reasoning}")

# 批量
results = go.batch_predict(['BTC', 'ETH', 'PEPE'])

# 扫描
top_coins = go.scan(min_score=60)

# 热点
hotspots = go.hotspots(limit=10)

# 机构
inst = go.detect_institutions('BTC')

# 优化参数
params = go.optimize('PEPE')

# 训练权重
go.train(iterations=100)
```

### 命令行
```bash
go-core predict BTC --interval 1h --period 90d
go-core scan --min-score 60
go-core hotspots
go-core detect BTC
go-core optimize PEPE
go-core train --iterations 100
```
