# go-pool - 撞球策略技能 v2.0

## 概述
撞球策略是一种资金轮转技术，模拟台球中的连续进球概念：
- **核心思想**: 让资金在不同币种间平滑流转
- **目标**: 始终持有正在上涨的币种，享受完整浪潮
- **原理**: 不同币种处于不同的回报周期阶段，系统自主识别并切换

## 策略版本

### v1.0 (G35)
- 基础周期检测
- Kelly仓位管理
- 20分钟冷却期

### v2.0 (G37)
- **动态止盈**: 12% + 波动率调整
- **快速止损**: 4%
- **go-core融合**: 评分权重25%
- **优化周期检测**: 更精准的阶段判断
- **加速扫描**: 45秒间隔
- **专注Top币**: BOME/TURBO/BONK/FLOKI/PEPE

## 周期阶段

| 阶段 | 描述 | 得分 |
|------|------|------|
| 盘整 | 横盘蓄力 | +10 |
| 启动 | 刚刚启动 | +25 |
| **加速** | **快速上涨 (主要进球区)** | **+35** |
| 高峰 | 接近顶点 | -15 |
| 衰退 | 开始下跌 | -35 |

|------|------|------|------|

| 止损 | 5% | 4% | 更快止损 |
| 扫描间隔 | 60秒 | 45秒 | 更高频率 |

## 状态机

```
┌─────────────────────────────────────────────────┐
│              撞球策略状态机                       │
├─────────────────────────────────────────────────┤
│                                                  │
│   盘整 → 启动 → 加速 → 高峰 → 衰退              │
│     ↓      ↓      ↓      ↓      ↓               │
│   买入   持有   重仓   止盈   切换              │

```

## 风险控制

1. **最大持仓**: 3个币种
2. **止损**: 4%硬止损
3. **止盈**: 动态12%+波动率
4. **冷却期**: 10分钟防频繁切换
5. **专注Top**: 只做高胜率币种
