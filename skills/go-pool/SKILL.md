# go-pool - 撞球策略 (Billiard Rotation Strategy)

## 概述
撞球策略是一种资金轮转技术，模拟台球中的连续进球概念：
- **核心思想**: 让资金在不同币种间平滑流转
- **目标**: 始终持有正在上涨的币种，享受完整浪潮
- **原理**: 不同币种处于不同的回报周期阶段，系统自主识别并切换

## 策略原理

### 回报周期理论
每个币种的涨跌形成一个完整的"周期"：
```
启动期 → 加速期 → 高峰期 → 衰退期 → 盘整期 → 下一浪
   ↓         ↓         ↓        ↓         ↓
  买入     持有      止盈     切换      观望
```

### 撞球切换逻辑
```
币种A (加速期) → 币种B (启动期) → 币种C (加速期) → ...
     ↓              ↓              ↓
  进球得分      进球得分      进球得分
```

## 核心算法

### 1. 周期阶段识别
```python
class CyclePhase:
    CONSOLIDATION = 0  # 盘整期 (蓄力)
    EARLY = 1          # 启动期 (刚启动)
    ACCELERATION = 2   # 加速期 (快速上涨) ← 主要进球区间
    PEAK = 3           # 高峰期 (接近顶点)
    DECLINE = 4        # 衰退期 (开始下跌)
```

### 2. 切换触发条件
- **主动切换**: 当前币进入衰退期，且发现更好的启动期币种
- **被动切换**: 当前币达到止盈点，利润已锁定
- **预防切换**: 发现高确定性启动信号，提前布局

### 3. 进球判断
```python
# 进球条件
is_scoring = (
    coin_phase == ACCELERATION and 
    momentum > 0.05 and
    rsi between 40-70 and
    cycle_completed > 0.3  # 已完成30%以上的上涨周期
)
```

### 4. 预算分配 (Kelly Criterion优化)
```python
# 基于周期位置动态分配
if phase == ACCELERATION:
    allocation = base * 1.5   # 加速期重仓
elif phase == EARLY:
    allocation = base * 1.2   # 启动期适度加仓
elif phase == PEAK:
    allocation = base * 0.5   # 高峰期减仓
else:
    allocation = base * 0.3   # 其他阶段轻仓
```

## 风险控制

1. **最大同持币种**: 3个 (避免分散)
2. **最低进球阈值**: 完成周期30%以上才切换
3. **止损保护**: 5%硬止损
4. **切换间隔**: 最少30分钟
