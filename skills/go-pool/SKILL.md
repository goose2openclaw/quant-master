# go-pool - 撞球策略技能 v3.0

## 概述
撞球策略是一种资金轮转技术，模拟台球中的连续进球概念。

## v3.0 整合架构
- **Top6 Meme动态交易**: RSI<35 + 动量确认
- **撞球周期轮转**: 盘整→启动→加速→高峰→衰退
- **go-core v2.0融合**: 300智能体预测
- **全自动自主交易**: 自主资产管理，自动调换币种

## Top6 Meme币种
- BOME, TURBO, BONK, FLOKI, PEPE, NEIRO

## 策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| KELLY_BASE | 30% | 基础仓位 |
| STOP_LOSS | 5% | 止损 |
| TAKE_PROFIT_BASE | 15% | 撞球止盈 |
| TOP6_TAKE_PROFIT | 20% | Top6止盈 |
| SCAN_INTERVAL | 30秒 | 扫描间隔 |

## Top6 Meme入场条件
- RSI < 35 (超卖)
- 动量 > 2%
- go-core买入确认

## Top6 Meme出场条件
- 止损: -5%
- 止盈: +20%
- 追踪止盈: 从高点回撤8%
- RSI超买: >70

## 撞球周期阶段

| 阶段 | 得分 | 操作 |
|------|------|------|
| 盘整 | +10 | 观望 |
| 启动 | +25 | 买入 |
| 加速 | +35 | 重仓持有 |
| 高峰 | -15 | 止盈 |
| 衰退 | -35 | 卖出切换 |

## 使用方法
```bash
python3 scripts/g37_pool.py
```
