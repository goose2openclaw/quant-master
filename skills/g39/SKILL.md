## 自主优化机制

### 自适应权重调整

```python
def optimize_weights(self):
    """根据表现自动调整策略权重"""
    for name in self.strategy_weights:
        history = self.get_strategy_history(name)
        
        # 计算胜率和收益率
        wins = sum(1 for h in history if h['pnl'] > 0)
        total = len(history)
        win_rate = wins / total if total > 0 else 0.5
        
        avg_return = sum(h['pnl'] for h in history) / total if total > 0 else 0
        
        # 调整权重
        if win_rate < 0.4 or avg_return < -0.05:
            self.strategy_weights[name] *= 0.8  # 降低
        elif win_rate > 0.6 and avg_return > 0.05:
            self.strategy_weights[name] *= 1.1  # 提高
        
        # 限制范围
        self.strategy_weights[name] = max(0.05, min(0.30, self.strategy_weights[name]))
```
