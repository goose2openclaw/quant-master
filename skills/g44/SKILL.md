# G44 v4.0 - 优化版本能化自主决策

## 版本
**v4.0** - 优化版本

## G43优化点

1. **提高买入阈值**: 0.02 → 0.05 (减少无效交易)
2. **LOT_SIZE自动修复**: 数量不足时自动增加50%
3. **优化止损逻辑**: 使用实际entry计算PnL
4. **突破市场检测**: 增加额外突破信号检测
5. **动态仓位管理**: 最大30%仓位

## 核心参数

| 参数 | 值 |
|------|------|
| 买入阈值 | >0.05 |
| 卖出阈值 | <-0.03 |
| 最大仓位 | 30% |
| 最小订单 | $5 |
| 扫描间隔 | 60秒 |

## 信号融合

```
综合信号 = (G44策略信号) x 60% + Polymarket x 40%
```

python3 skills/g44/g44.py
