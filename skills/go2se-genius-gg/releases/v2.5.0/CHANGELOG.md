# GO2SE Genius v2.5.0 更新日志

## 2026-05-06

### 🎯 核心更新: 插针全套系统 + 自主交易融合

#### 1. 插针捕获系统
- **向上插针做空**: 价格突破布林上轨+1%后做空
- **向下插针做多**: 价格跌破布林下轨-1%后做多
- **止损**: 插针极点外1%
- **止盈**: 入场价±2%
- **阈值**: 波动>1.5%触发，确认>0.5%

#### 2. 自主交易系统
- **三层监控**: 风险(1min) / 交易(5min) / 日报(21:00)
- **信号触发**: D>0.6 + RSI在40-70 + 成交量>1.2x
- **自主决策**: 信号强烈时自动操作，无需询问
- **安全护栏**: 保证金率<3.0谨慎，<2.5禁止开仓

#### 3. 声纳融合
- **6分制评分**: RSI/MACD/布林带/成交量/D分数/突破
- **多维度确认**: 避免单一指标误导
- **实时扫描**: 每5分钟全域名扫描

#### 4. 迭代学习
- **Hermes写入**: 每次决策自动记录
- **reflections.md**: 学习记录
- **promotions.md**: 版本迭代历史

## 技术规格

### 交易参数
- 杠杆: 动态 1x-5x (根据D分数)
- 止损: -1% (插针模式)
- 止盈: +2% (插针模式) / +20% (趋势模式)
- 保证金安全线: 3.0

### 扫描频率
- 风险监控: 1分钟
- 信号扫描: 5分钟
- 插针检测: 5分钟
- 日终报告: 21:00

### 决策规则

**买入信号**:
- D>0.6 + RSI在40-70 + Vol>1.2x
- 或 向下插针反弹确认

**卖出信号**:
- D<-0.3 或 RSI>85 或 跌破MA20
- 或 向上插针回落确认

**禁止开仓**:
- 保证金率 < 2.5
