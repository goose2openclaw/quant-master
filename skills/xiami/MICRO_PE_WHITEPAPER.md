# XIAMI 微私募 (Micro-PE) 平台白皮书

## 项目概述

**XIAMI 微私募** 是一个基于智能合约的加密货币量化投资平台，让普通投资者可以参与专业级量化交易策略。

---

## 核心理念

```
当前状态: XIAMI AI 机器人模拟交易测试迭代
     ↓
未来目标: 智能合约微私募平台
     ↓
资金流入: 公众投资 → 智能合约托管 → XIAMI策略执行 → 收益分配
```

## 智能合约机制

### 收益分配

```
投资 100% 
    │
    ├── 投资者: 75%
    │
    ├── 平台管理费: 5%
    │
    └── 绩效费: 20% (从收益中提取)
```

### 锁定期机制

| 参数 | 值 |
|------|-----|
| 锁定期 | 30天 |
| 赎回费 | 2% (提前赎回) |
| 正常赎回 | T+1 无费用 |
