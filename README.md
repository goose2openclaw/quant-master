# QuantMaster - QMT + vnpy 融合系统

## 目标
高精度克隆QMT功能，结合vnpy框架，构建完全自主可控的量化交易平台。

## 功能特性

### 1. 交易功能 (类QMT)
- 多账号管理
- 篮子交易
- 止损止盈
- 风控规则
- 快捷交易

### 2. 研究功能 (类vnpy)
- Alpha策略研究
- CTA回测
- 数据管理
- 因子分析

### 3. 架构
```
quant_master/
├── core/           # 核心引擎
│   ├── event.py    # 事件驱动
│   ├── gateway.py  # 交易所接口
│   └── engine.py   # 交易引擎
├── ui/            # 图形界面 (类QMT)
├── web_ui/        # Web界面
├── strategies/     # 策略库
├── backtest/      # 回测引擎
└── data/          # 数据管理
```
