## 工作流程

### 每日流程
```
1. 09:00 - 市场扫描
   - 获取24h行情
   - 检查持仓变化
   - 更新趋势判断

2. 任务执行
   - 空投任务检查
   - 趋势确认信号
   - 执行交易操作

3. 21:00 - 日报生成
   - 记录持仓
   - 计算收益
   - 优化策略
```

```

   ↓
2. 计算仓位
   ↓
3. 检查手续费优化
   ↓
4. 执行交易
   ↓
5. 验证结果
   ↓
6. 记录日志
```

```
绝对执行，不犹豫:
- 亏损5% → 立即止损

```

单一策略 ≤ 70%
