### 关键操作记录

#### 1. 全仓账户发现
- 原以为全仓账户为空 ($1)
- 实际发现逐仓账户有 $74 USDT + $50 各类币
- 全仓账户: $1 USDT, 总净值 ~$114
- API 划转限制: Cross Margin 无法直接从现货划转（报 Balanc No enough）

#### 2. 逐仓平仓 DOGE 成功
- 卖出 132 DOGE 逐仓 → $14.67 USDT
- 成功归集 $34.70 USDT 到现货

#### 3. Binance LOT_SIZE 和 NOTIONAL 限制
- XRP 持仓 $0.96 → 低于 $5 最低成交额，**无法卖出**
- ADA 持仓 $5.02 → 刚好低于 $10 最低额（对某些操作）
- ORCA 持仓 $4.62 → 低于 $10，无法市价卖出止损
- **教训**: 建仓时必须确保 NOTIONAL > $10

#### 4. 逐仓账户平仓 ETH/BNB/SOL 失败
- 报错 "Account has insufficient balance"
- **原因**: 这些逐仓对的 BTC 价值不足，无法开新仓位
- **教训**: 逐仓保证金必须高于 Binance 最低要求

#### 5. 5x杠杆买入 LINK
- 成功借入 $65 USDT
- 分次买入共 7.46 LINK (均价 ~$9.51)
- 保证金率 1.83，接近危险线
- **教训**: 杠杆仓位应控制在保证金率 > 3

#### 6. 自主卖出 DOGE 减仓
- 卖出 24 DOGE @ $0.112 = $2.69
- 决策理由: DOGE 仓位从 41% 降至 15%

### 关键教训清单

1. **NOTIONAL 限制**: 单笔订单必须 > $5-$10，否则市价单无法成交
2. **逐仓 vs 全仓**: 逐仓保证金不足时无法开新仓，但可以平仓
3. **杠杆控制**: 5x杠杆时保证金率 1.83 偏高，应保持在 3.0 以上
4. **资金归集**: 逐仓 USDT 可通过 ISOLATED_MARGIN→SPOT 划转
5. **跨仓划转**: SPOT→CROSS_MARGIN 有限制，需通过全仓 API 端点
6. **止损设计**: 仓位必须有足够的 notional value 才能执行止损
