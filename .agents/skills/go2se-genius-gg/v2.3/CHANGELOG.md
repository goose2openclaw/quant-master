# GG v2.3 更新日志

## 2026-05-05 v2.3

### 核心改进

1. **保证金率优先机制**
   - marginLevel < 3.0: 预警
   - marginLevel < 2.5: 强制减仓
   - marginLevel < 2.0: 全部清仓

2. **分批止盈策略**
   - 盈利>10%: 卖出30%
   - 盈利>15%: 卖出50%
   - 盈利>20%: 卖出70%

3. **RSI确认机制**
   - RSI>70 且 D>0: 谨慎买入
   - RSI<30 且 D<0: 谨慎卖出
   - RSI 40-60: D分数权重增加

4. **自动偿债**
   - 卖出后自动还款50%
   - 保持marginLevel稳定

### 决策矩阵

| marginLevel | D分数 | Mirofish | 操作 |
|-------------|-------|----------|------|
| <2.5 | 任意 | 任意 | 强制减仓 |

| >3.0 | D>0.8 | buy>60% | 5x买入 |
| >3.0 | D>0.5 | buy>40% | 3x买入 |
