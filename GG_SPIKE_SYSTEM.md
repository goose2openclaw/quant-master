# GG 插针全套系统

更新时间: 2026-05-06

---

## 系统概述

| 功能 | 描述 |
|------|------|
| 插针检测 | 监测1分钟内极端波动 |
| 做空机制 | 向上插针后做空 |
| 做多机制 | 向下插针后做多 |

---

## 核心机制

### 1. 插针检测

**触发条件**:
- 布林带 (20日, 2倍标准差)
- 价格突破上轨 +1% → 向上插针
- 价格突破下轨 -1% → 向下插针

**阈值配置**:
- 波动阈值: 1.5%
- 反转确认: 0.5%
- 止损: 1%
- 止盈: 2%

---

### 2. 做空机制 (向上插针)

**信号**:
```
价格突然上涨突破布林上轨
常见于: 流动性池止损、多头爆仓
```

**操作流程**:
```
1. 检测到向上插针 (涨破布林上轨+1%)
2. 等待价格回落确认 (回落>0.5%)
3. 入场做空 (卖出)
4. 止损: 插针高点上方1%
5. 止盈: 入场价下方2%
```

**示例**:
```
币价插针到 $10.50 (布林上轨 $10.20)
→ 回落至 $10.30
→ 做空，入场 $10.30
→ 止损 $10.61 (+3%)
→ 止盈 $10.09 (-2%)
```

---

### 3. 做多机制 (向下插针)

**信号**:
```
价格突然下跌突破布林下轨
常见于: 空头止损、主力吸筹
```

**操作流程**:
```
1. 检测到向下插针 (跌破布林下轨-1%)
2. 等待价格反弹确认 (反弹>0.5%)
3. 入场做多 (买入)
4. 止损: 插针低点下方1%
5. 止盈: 入场价上方2%
```

**示例**:
```
币价插针到 $9.80 (布林下轨 $10.00)
→ 反弹至 $9.90
→ 做多，入场 $9.90
→ 止损 $9.70 (-2%)
→ 止盈 $10.10 (+2%)
```

## 安全规则

| 规则 | 要求 |
|------|------|
| 保证金率 | 必须 > 3.0 |
| 单笔仓位 | 不超过总仓位 20% |
| 止损执行 | 必须执行，不扛单 |
| 每日交易 | 最多 3 笔 |
| 波动确认 | 需要二次确认 |

## 插针机会特征

### 高发时段
- 凌晨 00:00-04:00 (低流动性)
- 重大数据发布前后
- 交易所维护期间
- 极端行情波动时

### 高发币种
- 低流动性小币种
- 合约持仓量大的币种
- 刚上市的新币种

### 识别特征
- 成交量突然放大
- 价格瞬间波动 > 1%
- 快速收回大部分涨幅/跌幅
