## G25.3 终极版详细功能

### 1. 做多+做空机制
```
趋势向上 → 做多
趋势向下 → 做空
震荡市场 → 高抛低吸
```

### 2. T+0 即日买卖
```
支持同一天多次交易
快速止损止盈
日内收益锁定
```

### 3. 杠杆配置
```
主流币: 5x杠杆
Meme币: 10x杠杆
风险可控
```

### 4. 市场判断系统
```
TREND_UP: 追涨杀跌
TREND_DOWN: 杀跌追空
RANGE: 高抛低吸
VOLATILE: 突破动量
```

### 5. 分币种策略

**主流币 (稳健)**:
- RSI超卖<40买入
- 止损3%, 止盈15%
- 仓位10-15%

**Meme币 (激进)**:
- RSI超卖<35买入
- 止损5%, 止盈25%
- 仓位5-8%
