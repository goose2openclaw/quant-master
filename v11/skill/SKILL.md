# Q@C v11 Skill

## Name
quant-master-v11

## Description
QuantMaster Q@C v11 - 完全体加密货币量化交易系统。V8核心 + G12增补 + Mirofish仿真，三重权重分析，全模块打通，API完全接管Binance账户。

## Triggers
- `qcv11`, `quantmaster`, `交易系统`

## Actions

### Start Trading
启动Q@C v11交易系统
```
/qcv11 start
```

### Check Status
检查系统状态
```
/qcv11 status
```

### Stop Trading
停止交易系统
```
/qcv11 stop
```

## Options
- mode: LIVE/SIMULATE
- interval: 交易周期(秒)
- capital: 初始资金

## Version
11.0.0
