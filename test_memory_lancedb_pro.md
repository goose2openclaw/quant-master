## 技能概述
这是一个完整的OpenClaw插件维护技能，专门为维护和升级`memory-lancedb-pro`插件而设计。这是一个增强的LanceDB支持的长期记忆系统，具有：

### 核心特性
1. **混合检索** - 向量搜索 + BM25 + RRF融合
2. **交叉编码器重排序** - 提高检索质量
3. **多作用域隔离** - 5种作用域类型
4. **噪声过滤** - 自动过滤低质量记忆
5. **自适应检索** - 根据查询动态调整
6. **管理CLI** - 完整的命令行工具

## 实际应用场景

### 场景1: 智能记忆检索
```bash
# 使用CLI搜索记忆
memory-pro search "加密货币监控" --limit 10

# 导出记忆数据
memory-pro export --format jsonl --output memories.jsonl
```

### 场景2: 记忆质量优化
```bash
# 重新嵌入所有记忆
memory-pro reembed --all

# 迁移旧数据
memory-pro migrate --from builtin --to lancedb
```


### 步骤2: 配置插件
```json
{
  "memory": {
    "backend": "lancedb-pro",
    "lancedb": {
      "path": "/home/goose/.openclaw/memory/lancedb",
      "embedding": {
        "provider": "openai",
        "model": "text-embedding-3-small"
      }
    }
  }
}
```

### 步骤3: 初始化数据库
```bash
# 初始化LanceDB数据库
memory-pro init

# 导入现有记忆
memory-pro import --file memories.jsonl
```

```bash

memory-pro search "测试查询" --limit 5

# 验证检索质量
memory-pro stats --verbose
```
