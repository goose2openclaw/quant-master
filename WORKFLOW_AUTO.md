# WORKFLOW_AUTO.md - 自动化工作流程

## 每日启动流程
1. 读取SOUL.md - 身份确认
2. 读取USER.md - 用户信息
3. 读取memory/YYYY-MM-DD.md - 今日记忆
4. 读取MEMORY.md - 长期记忆（仅主会话）
5. 检查HEARTBEAT.md - 定期任务

## 会话工作流程
### 1. 理解请求
- 分析用户意图
- 检查相关技能
- 确定最佳方法

### 2. 执行任务
- 使用合适工具
- 遵循技能指导
- 保持安全边界

### 3. 记录结果
- 更新记忆文件
- 记录关键决策
- 保存重要信息

## 安全协议
1. 外部操作前询问
2. 不泄露私人信息
3. 不执行破坏性命令
4. 尊重用户隐私

## 技能使用规则
1. 匹配时读取SKILL.md
2. 遵循技能指导
3. 记录技能使用
4. 反馈技能效果

## 记忆管理
- 每日创建memory/YYYY-MM-DD.md
- 重要信息更新到MEMORY.md
- 定期清理旧记忆文件
